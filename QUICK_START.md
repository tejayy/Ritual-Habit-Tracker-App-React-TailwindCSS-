# Quick Start Guide

## 🚀 Getting Started in 5 Minutes

### 1. Installation
```bash
npm install
npm run dev
```
Open your browser to `http://localhost:5173`

---

## 📝 Creating Your First Ritual

1. Click the **"+ Add Ritual"** button (top right of center section)
2. Fill in the form:
   - **Name**: "Morning Meditation" (required)
   - **Description**: "10 minutes of mindfulness" (optional, supports **markdown**)
   - **Category**: "Morning" (optional)
3. Click **"Save Ritual"**

**Pro Tip**: Use markdown in descriptions:
- `**bold text**` for emphasis
- `*italic text*` for style
- `` `code` `` for technical notes

---

## ✅ Tracking Your Progress

### Mark as Complete
- Click the **circle** next to a ritual to check it off
- The circle turns orange with a checkmark ✓
- Ritual name gets a strikethrough

### Undo a Mistake
- Click the **"Undo"** button (appears after any action)
- Restores the previous state
- Works for the last 10 actions

### Celebrate Success 🎊
- Complete ALL rituals for the day
- Watch the confetti animation!
- Feel accomplished

---

## 🎯 Organizing Your Rituals

### Reorder with Drag & Drop
1. Hover over a ritual card
2. Click and hold the **≡ icon** (left side)
3. Drag up or down
4. Release to drop in new position

### Filter by Category
- Click any **category button** below the search bar
- Click **"All"** to see everything
- Categories are auto-generated from your rituals

### Search for Rituals
- Type in the **search bar**
- Searches both names and descriptions
- Results update in real-time

---

## 📦 Managing Rituals

### Archive (Hide Without Deleting)
1. Hover over a ritual card
2. Click the **archive icon** (📦)
3. Ritual is hidden but data is preserved
4. Click **"Show Archived"** to view archived rituals
5. Click archive icon again to restore

### Duplicate a Ritual
1. Hover over a ritual card
2. Click the **duplicate icon** (📋)
3. A copy is created with "(Copy)" in the name
4. Edit the copy as needed

### Delete Permanently
1. Hover over a ritual card
2. Click the **trash icon** (🗑️)
3. Ritual is permanently removed
4. **Note**: Only available for incomplete rituals

---

## 🖨️ Print Your Checklist

1. Click **"Print"** in the navigation bar
2. Review the print-friendly layout
3. Use your browser's print function (Ctrl+P / Cmd+P)
4. Print or save as PDF
5. Click **"Close Print View"** to return

**Use Case**: Print weekly checklists for offline tracking!

---

## 🌓 Toggle Dark/Light Mode

1. Click the **sun/moon icon** in the navigation bar
2. Theme switches instantly
3. Your preference is saved automatically
4. Works with system preference on first visit

---

## 📊 Understanding Your Stats

### Today's Progress (Right Sidebar)
- **Large Number**: Rituals completed today
- **Progress Bar**: Visual completion percentage
- **Reflection**: Dynamic message based on progress

### 14-Day Heatmap (Left Sidebar)
- **Each Square**: One day
- **Color Intensity**: More rituals completed = darker color
- **Hover**: Squares scale up for better visibility
- **Weekly %**: Your consistency for the past 7 days

### Streak Counter
- **🔥 Icon**: Appears on ritual cards
- **Number**: Consecutive days completed
- **Resets**: Breaks if you miss a day

## 💡 Pro Tips

### 1. Start Small
Begin with 3-5 rituals. Add more as you build consistency.

### 2. Use Categories Wisely
Organize by:
- **Time**: Morning, Afternoon, Evening
- **Area**: Health, Work, Personal
- **Type**: Physical, Mental, Social

### 3. Write Good Descriptions
Use markdown to add context:
```markdown
**Goal**: 10 minutes
*Why*: Reduces stress and improves focus
`Reminder`: Use Headspace app
```

### 4. Review Your Heatmap
Check your 14-day view weekly to spot patterns:
- Which days are hardest?
- Which rituals are you skipping?
- What's your overall trend?

### 5. Archive, Don't Delete
Keep your history by archiving rituals you're taking a break from.

### 6. Celebrate Wins
Completing all rituals triggers confetti - enjoy it!

### 7. Print Weekly
Generate a physical checklist every Monday for offline tracking.

## 🆘 Troubleshooting

### Rituals Not Saving?
- Check browser console for errors
- Ensure localStorage is enabled
- Try refreshing the page

### Drag & Drop Not Working?
- Make sure you're clicking the ≡ icon
- Try on a different browser
- Check if touch events are enabled (mobile)

### Confetti Not Showing?
- Complete ALL active rituals (not archived ones)
- Check if animations are enabled in browser
- Try refreshing the page

### Dark Mode Not Persisting?
- Check if localStorage is enabled
- Try manually toggling again
- Clear browser cache if needed
