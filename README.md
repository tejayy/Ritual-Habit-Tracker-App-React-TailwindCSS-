# LifeOS - Daily Rituals Tracker

A beautiful, modern habit tracking app to help you build consistency through daily rituals.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

## ✨ Features

### Quick Wins
- 🔍 **Search & Filter** - Find rituals instantly
- 📦 **Archive System** - Hide without deleting
- ↩️ **Undo Button** - Fix mistakes easily
- 🎊 **Confetti Animation** - Celebrate completions
- 📝 **Markdown Support** - Rich text descriptions
- 📋 **Duplicate Rituals** - Copy existing ones
- 🖨️ **Print View** - Physical checklist option

### Phase 1 Features
- 🏷️ **Categories/Tags** - Organize your rituals
- 🎯 **Drag & Drop** - Custom ordering
- 📊 **Weekly Statistics** - Track consistency
- 🔥 **Streak Tracking** - Build momentum

### Design
- 🌓 **Dark/Light Mode** - System preference detection
- 🎨 **Modern UI** - Beautiful terracotta/orange theme
- 📱 **Responsive** - Works on all devices
- ✨ **Smooth Animations** - Polished interactions

## 📖 Usage

1. **Add Rituals**: Click "+ Add Ritual" and fill in details
2. **Organize**: Drag to reorder, assign categories
3. **Track**: Check off rituals as you complete them
4. **Review**: View your 14-day heatmap and weekly stats
5. **Print**: Generate a physical checklist anytime

## 🛠️ Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS 4
- DnD Kit (drag & drop)
- Canvas Confetti
- React Markdown
